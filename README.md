# Сокращение ссылок

## Лабораторная работа № 4

**Вариативная часть №9:** Изменение длинной ссылки для существующего короткого кода.


## Описание

Программа преобразует длинные ссылки в короткие коды и позволяет управлять ими.

## Функции
__init__ — создаёт пустые хранилища для ссылок.

_generate_code — делает из длинной ссылки короткий код (6 символов).

add_url — сохраняет новую ссылку или возвращает старый код, если ссылка уже есть.

get_long_url — ищет длинную ссылку по короткому коду.

exists — проверяет, существует ли короткий код.

get_all_links — показывает все сохранённые ссылки.

update_long_url — меняет длинную ссылку для существующего короткого кода (это задание №9).

main — выводит меню и связывает действия пользователя с нужными функциями.
